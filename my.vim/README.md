## Custom shortcuts

### Buffer navigation

* Next buffer: `\ + RIGHT`
* Previous buffer: `\ + LEFT`

## Tips

### Vim Shortcuts

Mode visual selection

* Select line: V
* Select range: v
* copy: y
* paste after cursor: p
* undo: v
* redo: Ctrl + R

### General tips

* close buffer: :bd
